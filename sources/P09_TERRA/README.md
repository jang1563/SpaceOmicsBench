# P09: Telomeric RNA (TERRA)

- **Paper**: "Telomeric RNA (TERRA) increases in response to spaceflight and high-altitude climbing"
- **Journal**: Communications Biology, June 2024
- **DOI**: 10.1038/s42003-024-06014-x
- **Mission**: Inspiration4 + NASA Twins Study + Mt. Everest climbing expedition

## Data Types

- scRNA-seq / Multiome (I4 PBMCs)
- Bulk RNA-seq (Twins Study, Everest)
- K-mer motif analysis (TERRA UUAGGG repeats)

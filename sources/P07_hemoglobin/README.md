# P07: Hemoglobin Gene Expression in Space

- **Paper**: "Spatiotemporal expression and control of haemoglobin in space"
- **Journal**: Nature Communications, June 2024
- **DOI**: 10.1038/s41467-024-49289-8
- **Mission**: Inspiration4 + JAXA CFE + NASA Twins Study (cross-mission)

## Data Types

- Bulk RNA-seq (Twins Study)
- cfRNA-seq (JAXA CFE, 6 astronauts, 11 timepoints)
- miRNA-seq (Twins Study)
- Single-cell transcriptomics (I4 PBMCs)

- **OSD-570**: I4 PBMCs

## Key for Benchmark

Critical cross-mission comparison paper. Source Data contains hemoglobin gene
expression data across all three missions, enabling cross-mission concordance tasks.
Twins Study data here is aggregate/summary level (no individual identifiers).

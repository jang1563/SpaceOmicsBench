# Spatial Transcriptomics (Skin)

Inspiration4 skin punch biopsy spatial gene expression (NanoString GeoMx).

## Sources
- **OSD-574**: GeoMx Digital Spatial Profiler processed DCC files, normalized counts
- **P06 Supp Data 1**: DEG tables (95 up, 121 down genes)
- **P06 Supp Data 2**: Pathway enrichment (FGSEA, GSVA, MSigDB Hallmark)
- **P06 Source Data**: All figure data (26.6 MB)

## Key Numbers
- 18,676 genes across 95 regions of interest (ROIs)
- Whole Transcriptome Atlas (WTA) probe panel
- Timepoints: L-44 (pre-flight) vs R+1 (post-flight)
- Tissue: Deltoid skin punch biopsies

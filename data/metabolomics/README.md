# Metabolomics

Inspiration4 plasma untargeted metabolomics.

## Sources
- **OSD-571**: Plasma metabolomics (ANP + RP chromatography + MS)
- **P02 Source Data**: Differential metabolite results

## Key Numbers
- 1,135 metabolites total: 865 aqueous normal phase (ANP) + 270 lipid (reverse phase)
- Metabolites matched to in-house standards (mass, RT, MS/MS fragmentation)
- Statistical model: limma (~astronaut + flightStatus)
- Significance: adj. p < 0.05, |logFC| > 1

# Proteomics

Inspiration4 plasma proteomics and extracellular vesicle/particle (EVP) proteomics.

## Sources
- **OSD-571**: Plasma proteomics (Seer Proteograph + LC-MS/MS DIA) + EVP proteomics (LC-MS/MS DDA)
- **P02 Source Data**: Differential abundance results (limma model)

## Key Numbers
- Plasma: 2,992 unique proteins detected, 1,765 after filtering
- EVP: 1,443 unique proteins detected, 527 after filtering
- Statistical model: limma (~astronaut + flightStatus)
- Significance: adj. p < 0.05, |logFC| > 1

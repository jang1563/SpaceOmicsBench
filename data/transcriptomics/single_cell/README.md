# Single-Cell / Single-Nucleus Multi-ome

Inspiration4 PBMC single-nucleus RNA-seq + ATAC-seq (10x Genomics Multiome).

## Sources
- **OSD-570**: Cell Ranger ARC v2.0.0 processed outputs (GEX + ATAC matrices)
- **OSD-570**: Cell Ranger v6.1.1 V(D)J outputs (TCR/BCR clonotypes)
- **P03 Supp Data 1-15**: Pathway enrichment, TF motifs, drug candidates

## Key Features
- ~151,411 PBMCs across 9 gross cell types
- Simultaneous gene expression + chromatin accessibility
- TCR and BCR immune repertoire data
- 6 timepoints (L-92 to R+82)

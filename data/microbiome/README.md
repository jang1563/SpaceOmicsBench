# Microbiome

Inspiration4 metagenomics and metatranscriptomics from multiple body sites.

## Sources

### Human Microbiome
- **OSD-572**: Skin, oral, nasal swab metagenomics + metatranscriptomics
- **OSD-630**: Stool metagenomics (gut microbiome)
- **P05 Supp Data**: Species abundance, diversity metrics
- **P06 Supp Data 3**: Skin microbiome taxonomy (826 bacterial, 9,819 viral species)

### Environmental
- **OSD-573**: Dragon capsule surface metagenomics

## Interactive Portal
- I4 Microbiome Browser: https://soma.weill.cornell.edu/apps/I4_Microbiome/
